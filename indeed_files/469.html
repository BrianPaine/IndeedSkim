SQL Report Analyst

Richmond, Virginia

About the role:

UDig is seeking a SQL Report Analyst to support our client located in Richmond, Virginia. The ideal candidate will bring the ability to thrive in a highly collaborative and fast-paced work environment. The role is responsible for designing and developing reports primarily using SQL Server Reporting Services (SSRS). The candidate is also responsible for adhering to our client's development best practices. In addition, supporting and contributing to our client's SDLC and Agile development process is an essential component of the role.

What you’ll do:

  • Adhere to professional software engineering practices & best practices for the full software development life cycle, including coding standards, code reviews, source control management, build processes, testing, and operations.
  • Analyze business requirements and produce detailed technical designs for innovative software solutions.
  • Work with the business community to gather requirements and create designs for new reports or changes to existing reports.
  • Create report requirements and design documents that follow project standards and articulates the solution to develop on an appropriate level, leveraging templates where possible and analyzes the impact to existing systems and alternatives
  • Conduct walkthroughs of design documents and work with the team to address comments resulting from the reviews.
  • Determine the impact and potential costs of proposed changes
  • Create technical training documentation regarding the current function(s) of new or existing reports
  • Provide technical knowledge transfer for new or existing reports to other technical staff
  • Create new reports and enhance existing reports, primarily using SQL Server Reporting services (SSRS) using secure coding practices
  • Produce ad-hoc reports and/or SQL scripts when requested
  • Work with and/or modify the SQL Server-based data model as required.
  • Write queries, views or stores procedures to extract data from the SQL Server database.
  • Provide support of the SQL Server database as needed
  • Follow established reporting standards and configuration management procedures
  • Troubleshoot any issues with reports, data load failures or data retrieval issues
  • Provide performance tuning on existing reports to enhance end user experience
  • Provide support to the technical staff, internal users and external users in the support and use of new or existing SSRS reports and ad-hoc SQL scripts.
  • Provide general production support for new and/or existing reports.
  • Possess and applies advanced knowledge of technology principles, best practices and procedures to the completion of complex software development assignments
  • Ensure that all code, configuration data, and system documentation is accurate, complete, understandable, and adheres to our client's standards
  • Create unit test cases to appropriately verify new or modified reports, executes those scripts and resolves identified issues to ensure comprehensive testing of business software using non-sensitive data
  • Support the reports testing process by the user community. Assigned system incidents and defects identified during the testing process should be fixed, unit tested and checked into the source control system.
  • Gather or creates test data necessary to execute test scripts in compliance with our client's standards. 
  • Execute tests and documents results.
  • Work with QA on test results, defect remediation and script re-use.

What you need:

  • Report and database development and unit testing
  • Report design
  • Technical reports training
  • Report maintenance and support
  • Ascertain and meet customer expectations
  • Work effectively and independently in a fast paced team environment where priorities can rapidly change
  • Prioritize own work activities with minimal guidance and coaching
  • Manage competing priorities to meet goals
  • Communicate effectively orally and in writing
  • Develop and manage detailed and accurate work plans and appropriately communicate work plan risks and impacts to management
  • Provide, maintain and follow technical documentation
  • Software development methodologies and best practices
  • Secure coding practices
  • SQL Server
  • SQL Server Reporting Services (SSRS)
  • SQL Service Analysis Services (SSAS) – preferred
  • SQL Server Integration Services (SSIS) - preferred
  • Candidates must have a Bachelor’s degree from a four-year college or university in computer science or closely related field and 4 or more years of experience. Preferred experience includes 4 years of System Development Life-Cycle experience working with large, complex systems and two years’ experience working on Agile Scrum teams. This position will be on a contract basis for a period of up to 12 months with the potential for extension.
