                JOB DESCRIPTIONPiper Companies is looking for a Data Scientist who is creative, innovative and intellectually curious who is interested in joining a local Healthcare Analytics team for an exciting opportunity to work in one of the world&#39;s leading, large-scale data insights team.

Responsibilities of the Data Scientist:


	You will play an important part in designing and delivering statistical / machine learning studies and predictive analytics solutions in a range of challenging analytical areas. &nbsp;&nbsp;
	This team works hand-in-hand with statisticians, epidemiologists and disease area experts across the wider global team, leveraging a vast variety of anonymous information from sources such as electronic health records.


&nbsp;Qualifications of the Data Scientist:

	Excellent knowledge of supervised machine learning methods.
	Strong programming skills in Python and/or R. Experience in SPARK or Scala is highly beneficial.


Compensation of the Data Scientist:


	Competitive salary + bonus potential
	Stimulating, Innovative, Fun &amp; friendly global work environment

	Great benefits including great office culture, healthcare &nbsp;and competitive compensation package

    function checkAuth(immediate,callBackFunction) {
        gapi.auth.authorize(

            callBackFunction);
    }
    /**
     * Called when authorization server replies.
     *
     * @param {Object} authResult Authorization result.
     */
    function handleAuthResult(authResult) {
        if (authResult && !authResult.error) {
            var token = gapi.auth.getToken();
            $("#g_auth_token").val(token["access_token"]);
            callPicker(token);
            if(checkInterval!=undefined){
                clearInterval(checkInterval);
            }
        }
    }
    /**
     * Called when authorization server replies. and only Triggered on Page Load
     *
     * @param {Object} authResult Authorization result.
     */
    function handleAuthResultOnLoad(authResult) {
        if (authResult && authResult.error) {
            // No access token could be retrieved, force the authorization flow.
            $("#drive-chooser").click(function(){
                checkAuth(false,handleAuthResult);
                //on Chrome IOS

            $("#drive-chooser").click(function(){
                checkAuth(true,handleAuthResult);

    // Use the Google Loader script to load the google.picker script.
    function callPicker(AUTH_TOKEN) {
        google.load('picker', '1', {"callback" : createPicker});
    }
    // Create and render a Picker object for searching images.
    function createPicker() {
        var AUTH_TOKEN = $("#g_auth_token").val();
        var picker = new google.picker.PickerBuilder().
            addView(google.picker.ViewId.DOCUMENTS).
            addView(google.picker.ViewId.PDFS).
            setOAuthToken(AUTH_TOKEN).
            setCallback(pickerCallback).
            build();
        picker.setVisible(true);
    }
    // A simple callback implementation.
    function pickerCallback(data) {
        var url = 'nothing';
        if (data[google.picker.Response.ACTION] == google.picker.Action.PICKED) {
            $(".cloud-btn .big-buttons").hide();
            var doc = data[google.picker.Response.DOCUMENTS][0];
            ID = doc["id"];
            gapi.client.load('drive','v2', function() {
                var request = gapi.client.drive.files.get({
                    'fileId': ID,
                    "projection" : "FULL"
                });
                request.execute(function(response) {
                    console.log(response);
                    $("#cloud").val("google");
                    if(response.downloadUrl){
                        var fileName = response.title;
                        $("#fileName").val(fileName);
                        $("#fileUrl").val(response.downloadUrl);
                    }else{
                        var fileName = response.title+".pdf";
                        $("#fileName").val(fileName);
                        $("#fileUrl").val(response["exportLinks"]["application/pdf"]);
                    }
                    $("#message-file").fadeOut('slow');
                    $("#resume_name span").html(fileName);
                    $("#resume_name").css("display","block");
                    $(".cloud-upload-holder .big-buttons").hide();
                    $('.attach-resume-title, #fakeDivBtn #chooseBtn, #hideAttach, #real, .notes').hide();
                });
            });
        }
    }
