		The SQL Data Analyst works closely with the Director of IT and interfaces with users and managers at all levels to provide useful information from data sources.&nbsp; The primary function of the position writing is SQL queries and creating and modifying SSRS reports. Excellent Excel skills are also required.

&nbsp;ESSENTIAL DUTIES AND RESPONSIBILITES



Develop / modify / run SQL queries to access, report on, modify, and upload data in SQL databases.

Run SQL queries to research issues, identify data entry errors, and update data.

Develop / modify / run standard reports utilizing Microsoft SQL Server Reporting Services.

Create, modify, and manipulate Excel spreadsheets.

Develop / modify / run ad hoc and standard reports utilizing Crystal Reports.

Recognize, troubleshoot and resolve Infor Visual Enterprise Resource Planning (ERP) software data and related issues / anomalies.

Assist Visual ERP users with a wide variety of issues relating to database application functionality.

Test issues in a test database as a step to resolution, and to recognize when it is prudent to use a test environment.

Perform routine tasks such as running software utilities.

Assist with projects including software implementation, adding application functionality, and department initiatives.

Work with technical support and outside consultants.

Set up, manage, modify and remove user accounts.

Develop constructive and cooperative working relationships with others and maintain them over time.

Identify opportunities for process improvement and increased efficiency.

Follow work rules and standard safety regulations at all times.

Perform other duties as assigned.



&nbsp;QUALIFICATIONS



U.S. Person status required due to export control.

Possess an understanding of database structure and concepts.

Skilled in using Microsoft SQL Server Management Studio.

Skilled in using Microsoft SQL Server Reporting Services.

Advanced computer literacy in Windows environment including MS Office (Excel, PowerPoint, etc.).

Excellent interpersonal and communication skills, team oriented, and ability to deal with a wide variety of personalities and learning styles.

Ability to train and / or give presentations to individuals and groups.

Ability to define problems, collect data, establish facts, and draw valid conclusions.

Able to make independent decisions based on input.

Excellent organizational skills.

Ability to work off hours including weekends and holidays if needed.

Ability to travel within the Puget Sound Region.



&nbsp;EDUCATION/EXPERIENCE



U.S. Person status required due to export control.

Three years of experience developing SQL queries.

Microsoft SQL Server Reporting Services experience desired.

Advanced Excel experience.

Three years of increasing responsibility utilizing database applications (front-end or back-end).

Associates degree from a two-year college or university in a similar field preferred.

Experience developing reports with software such as Crystal Reports desirable.

Experience training users both individually and in a classroom environment helpful.

Excellent interpersonal and communication skills and ability to deal with a wide variety of personalities and learning styles.
